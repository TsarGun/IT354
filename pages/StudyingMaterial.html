Advanced Array Methods

The logic is that the Higher-Order Function creates a copy of massive and then work with it.

Map - changes all of the elements of the massive and then return the massive.

Filter - filters elements of the massive by some criteria and then returns filtered massive of the elements.

Find - find the first element which passed the condition. Returns element.

Every - if all of the element has passed criteria returns True, if not - False

Some - if at least one element passed the condition returns True, if none - False

ForEach - apply the function to each array element

Reduce - this method executes a reducer function (that you provide) on each element of the array. It also has an accumulator(result) that accumulates callback's return values. If the accumulator isn't stated it's equal to the 0 element of the array. The result is a single output value.

IndexOf - method returns the first index at which a given element can be found in the array, or -1 if it is not present.
